    PDF creator endpoints.

    Methods

    createPDFByOrders(enterpriseKey, filters) → {Response}

    This endpoint allows to generate tickets using the standar Chazki format.

    The file to be generated is in PDF extension.

    URL: https://us-central1-chazki-link-dev.cloudfunctions.net/createPDFByOrders

    Parameters:
    Name Type Description
    enterpriseKey String

    Unique authorization code that identifies the customer.
    filters Object

    Object with filters to generate tickets in PDF format.

    Properties
    Name Type Description
    trackCodes Array.<String>

    Order Tracking code.
    Throws:

    If the enterpriseKey or the ticket correct name is not provided, the PDF will not be generated.

    Returns:
    Response -

    Object with the PDF URL address.

    Examples

    Parameters sent inside the request body in JSON format.

     {
   "enterpriseKey": "XXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX",
   "filters": {
     "trackCodes":["XXXXXXXX","YYYYYYYYYY"]
   }
 }

    Response

     {
   "success": true,
   "file": "https://storage.googleapis.com/download/storage/v1/b/xxxxxxxxxxxxx/yyyyyyy"
 }
